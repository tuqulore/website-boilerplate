# Getting Started

This chapter covers everything you need to launch a new site with the boilerplate: prerequisites, scaffolding, editing the first page, and the shape of the resulting project.

- [Installation](/en/getting-started/install/): prerequisites and how to scaffold from the CLI.
- [Write your first page](/en/getting-started/first-page/): replace the placeholder content in the scaffolded project.
- [Project structure](/en/getting-started/structure/): what each scaffolded directory is for.

If this is your first time with the boilerplate, read them in order starting from installation.
