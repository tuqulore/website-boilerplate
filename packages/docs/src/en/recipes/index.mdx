# Recipes

This chapter collects the common scenarios you hit while building sites with the boilerplate, arranged as one page per scenario, each readable on its own.

- [Tailwind CSS](/en/recipes/tailwind/): default setup and how to add Tailwind plugins.
- [Island granularity](/en/recipes/island/): deciding what to hydrate and at which granularity.
- [Adding i18n](/en/recipes/i18n/): directory split for locales, per-locale navigation, and translation fallbacks.
- [Adding a plugin](/en/recipes/plugin/): registering third-party Eleventy plugins alongside the preset.
