# @tuqulore-inc/website-boilerplate Documentation

This site documents the design and usage of @tuqulore-inc/website-boilerplate.
It has four sections:

- **Getting Started** — prerequisites and how to launch a first page.
- **Architecture** — the design of the preset, Preact, Partial Hydration, and PostCSS layers.
- **Recipes** — practical patterns arranged by scenario.
- **Roadmap** — planned work and past architecture decisions (ADRs).

Each section is currently under construction.
