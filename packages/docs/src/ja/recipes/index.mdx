# Recipes

本章は、本ボイラープレートで実装するときによく直面するシナリオを、レシピ集としてまとめます。1 ページ 1 シナリオで、独立して読めるように書きます。

- [Tailwind CSS](/ja/recipes/tailwind/): Tailwind のセットアップと、プラグインを増やす手順を扱います。
- [Island の粒度](/ja/recipes/island/): どこを Island 化するか、どのトリガーで hydrate するかの判断基準を扱います。
- [i18n の足し方](/ja/recipes/i18n/): 複数言語のディレクトリ分割、ナビゲーションの分岐、フォールバックの流れを扱います。
- [プラグインを追加する](/ja/recipes/plugin/): 既存の Eleventy プラグインをプリセットに載せる手順を扱います。

各レシピは Architecture 章の設計判断を踏まえた実装ガイドです。背景を先に把握したい場合は [Architecture](/ja/architecture/) を参照してください。
