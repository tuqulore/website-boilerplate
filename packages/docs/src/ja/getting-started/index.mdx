# Getting Started

本章は、本ボイラープレートで新しいサイトを立ち上げるまでの手順を扱う。前提から scaffold、初回の編集、プロジェクト構成の把握までを 3 ページに分ける。

- [インストール](/ja/getting-started/install/): 動作環境の準備と、CLI で scaffold する手順。
- [最初のページを書く](/ja/getting-started/first-page/): scaffold されたプロジェクトで、開発サーバを起動して 1 枚目のページを書き換える。
- [プロジェクト構成](/ja/getting-started/structure/): scaffold が用意するディレクトリの役割。

初めて本ボイラープレートに触るなら、インストールから順に読むのが早い。
