# Getting Started

本章は、本ボイラープレートで新しいサイトを立ち上げるまでの手順を扱います。前提から scaffold、初回の編集、プロジェクト構成の把握までを 3 ページに分けます。

- [インストール](/ja/getting-started/install/): 動作環境を整え、CLI で scaffold する手順を扱います。
- [最初のページを書く](/ja/getting-started/first-page/): scaffold されたプロジェクトで開発サーバを起動し、1 枚目のページを書き換えます。
- [プロジェクト構成](/ja/getting-started/structure/): scaffold が用意するディレクトリの役割を扱います。

初めて本ボイラープレートに触る場合は、インストールから順に読み進めるのが早道です。
