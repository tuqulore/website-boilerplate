# Website Boilerplate ドキュメント

本ドキュメントは Website Boilerplate の設計と使い方を扱います。以下の 4 章で構成します。

- [Getting Started](/ja/getting-started/): 動作環境の準備、最初のページの立ち上げ、ディレクトリ構成の把握までを扱います。
- [Architecture](/ja/architecture/): 本ボイラープレートの全体設計の見取り図です。
- [Preset の規約](/ja/preset/): preset を使ってサイトを書くときの規約集です。
- [Plugins](/ja/plugins/): 各プラグイン単位の内部設計と応用ケースを扱います。
